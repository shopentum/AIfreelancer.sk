# Kanban Dashboard

Osobný execution board. Špecifikácia: [`SPEC_IMPLEMENTATION.md`](./SPEC_IMPLEMENTATION.md), fázy: [`PHASES.md`](./PHASES.md).

## Vývoj

```bash
cd apps/kanban
npm install
npm run dev
```

App beží na http://localhost:5174

## Build

```bash
npm run build
npm run preview
```
